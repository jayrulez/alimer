**Alimer** is a cross-platform 2D and 3D game engine implemented in C++17, inspired by [Urho3D](https://github.com/urho3d/Urho3D)

## Features

- Cross-platform.
- Modern graphics-rendering API using Vulkan, Direct3D12 and Direct3D11.
